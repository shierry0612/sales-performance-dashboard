# Sales Performance Analytics Dashboard

## Objective
Build an interactive Power BI dashboard to monitor sales performance and support data-driven decision-making.

## Tools
- Power BI
- Excel / CSV

## Key Metrics
- Total Revenue
- Profit and Profit Margin
- Revenue by Region, Province, and Store
- Time-based performance trends

## Key Features
- Interactive filters by region, store, and date
- Top and bottom performing stores analysis
- Geographic visualization
- Data cleaning and validation
